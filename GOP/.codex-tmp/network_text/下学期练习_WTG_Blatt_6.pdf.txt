Aufgabe 1
Für x ∈ R definiere das Dirac-Maß δx auf R über δx (A) = 1 falls x ∈ A und δx (A) = 0 sonst.
Zeige:

  1. δx besitzt keine Dichte bezüglich des Lebesgue-Maßes.
     Hinweis: Nutze den Satz von Radon-Nikodým.

  2. Sei (Ω, F, P) ein Wahrscheinlichkeitsraum. Für f : Ω → R, ω 7→ x gilt δx = Pf wobei

                           Pf (A) = P(f −1 (A)) = P({ω ∈ Ω : f (ω) ∈ A})

     das pushforward Maß (bzgl. P und f ) ist.
