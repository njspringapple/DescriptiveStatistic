# 考前辅导25 题解 Markdown

本目录由 `考前辅导25` 下的 PDF 整理而来，目标是供 Obsidian 直接阅读。

## 内容

- 每个 PDF 对应一个同名 Markdown 题解文件。
- 数学公式统一整理为 Obsidian 兼容的 `$...$` 或 `$$...$$`。
- 题目按 `## Aufgabe ...` 作为主计数单位。
- 图形题保留题面分析要点，不臆造 PDF 图中没有给出的数值。

## 核对文件

- `AUDIT.md`：文件数、题数、残留 OCR 检查。
- `THEMEN_INDEX.md`：按主题归类后的题目数量汇总。
